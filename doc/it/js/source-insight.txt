Add the following regexp to customize the parsing for "method":

- Option->Document Options
- Language: javascript
- custom tag type: method
- custom pattern: ^\w*function\w+\([_a-zA-Z0-9]+\)\w*(

This enables SI to list the private functions inside a IIFE function body.


To add more custom parsing command, select "Language Properties" from context menu (right-click), and choose the "Custom Parsing" tab for adding more than one parsing commands.
