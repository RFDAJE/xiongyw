3. HOME environment & Init file

This is to set change the HOME environment to point to the "doc/it/emacs" subdirectory of the local git workspace.
How Emcas find Init file: "http://www.gnu.org/software/emacs/manual/html_node/emacs/Find-Init.html#Find-Init": 
here we stick to "..\.emacs.d\init.el", not using ".emacs" or ".emacs.el".

- Launch emacs by double clicking "D:\emacs\emacs-24.2\bin\runemcas.exe" (or create a shortcut and drop it on the task bar)
- Type: "M-x getenv RET HOME RET" to get the default HOME environment variable value, e.g. "C:\Users\xiongyw\AppData\Roaming". 
- Go there to have a look, a new directory is created by emacs: "..\Roaming\.emacs.d\".
- Create a new file "..\Roaming\.emacs.d\init.el", and put the
- following line to change the HOME and manually load the real "init.el", the "rabbit-hole":

  (setenv "HOME" "d:/github/xiongyw/doc/it/emacs")
  (load-file "~/.emacs.d/init.el")




- Type "M-x apropos RET process-environment RET" to get a full list of environment variables.
- "describe-variable": C-h v variable-name RET
- M-: (message "%s" variable-name)
