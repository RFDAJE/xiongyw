0. Summary
This is a memo on how to play a TS stream by using the DekTec card on Linux.

For playing TS streams, DekTec provides:

StreamXpress, for Windows, in binary/installer form
DtPlay, for Linux, in source code form
 

On Linux, DtPlay has two dependencies:

The driver for the Dektec cards. The driver is a kernel module, which will create device nodes once loaded. 
The DTAPI header file (DTAPI.h) and libraries (DTAPI.o or DTAPI64.o).
 

The following briefly describes how to get these stuff in place.

7. Output Signal Strength
"-ml" controls the output level, and by default, it's "-27.5 dBm". 

Using the signal checking equipment (PROMAX TV Explorer HD, in Beijing Lab) to measure the direct output of the signal strength:

"-ml -35.0": > 70.2 dBuV
"-ml -30.0": > 75.2 dBuV
"-ml -27.5": > 78.0 dBuV
"-ml -20".0: > 85.9 dBuV
"-ml -15.0": > 90.8 dBuV
"-ml -10.0": > 95.7 dBuV
