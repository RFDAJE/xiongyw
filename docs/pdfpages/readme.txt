print pdf into booklet using A4 paper
-------------------------------------
bruin, 2012-06-10


1. assumptions:

- using A4 paper (21x29.7cm), which is regarded as 16 kai
- signature size is 4 for glue binding (for stitch binding,
  the signature size could be 24 or so).

2. to print into 32k booklet:

- using the "32k.tex" as the template

3. to print into 64k booklet:

- using the "32k.tex" and then followed by "64k.tex" template: 
  yes, using a 2-pass process.
